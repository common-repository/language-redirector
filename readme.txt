Plugin that performs a country-based redirect

== Description ==

This plugin allows automatic redirection to your country of origin

* [WordPress Language Redirector] (http://www.webcreates.eu/download/language-redirector.zip)

== Premium version ==

The paid version allows you to enter redirects without the current limit to two countries
Buy link: http://www.webcreates.eu/sell_plugin.php

== Installation ==

= Installation Instructions =

1. Go to Plugins > Add New > Search
2. Type Language Redirector in the search box and hit Enter
3. Click on the button Install and then activate the plugin

= Manual Installation =

The manual installation method involves downloading our plugin and uploading it to your webserver via your favourite FTP application.

The WordPress codex contains [instructions on how to do this here]:
http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation
